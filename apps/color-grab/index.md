## 目の前の色を、そのままパレットに

カメラをかざすだけで、目の前にあるモノの色をリアルタイムに読み取り、HEX / RGB / HSL のカラーコードと色名で採取できる iOS アプリです。

## 主な機能

- **カメラでライブ採取**: 画面中央の色をリアルタイム表示。シャッターで色を固定して、そのままパレットに保存
- **写真から採取**: 保存済みの写真をタップして、好きな場所の色を抽出
- **カラーコード表示**: HEX / RGB / HSL に対応。ワンタップでコピー
- **色名がわかる**: CSS カラーネームと JIS 慣用色名の辞書から、いちばん近い色名を自動判定
- **パレット管理**: 採取した色をパレットにまとめて整理。PNG 画像として書き出し可能

## プライバシー

採取した色やパレットはすべて端末内にのみ保存されます。カメラ・写真は色の抽出のためだけに端末内で処理され、外部へ送信されることはありません。
