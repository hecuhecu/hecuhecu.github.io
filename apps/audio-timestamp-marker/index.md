## 長い録音から「あの発言」をすぐ探せる

インタビューや会議の音源を波形表示で再生しながら、気になった瞬間にワンタップでマーカーを打てる iOS アプリです。取材ライター・記者・議事録担当者のための、聞き直し時間をなくすツールです。

## 主な機能

- **波形再生 + ワンタップマーカー**: 音声ファイルを読み込み、波形を見ながら再生中にマーカーを記録
- **メモ + カラータグ**: マーカーに自由記述メモと3色タグ（重要 / 要確認 / 引用候補）を付与
- **一覧 + タグフィルタ**: マーカーをタイムスタンプ順に一覧表示し、タグで絞り込み
- **書き出し**: マーカー一覧を CSV / SRT / Markdown 形式でエクスポート

## プライバシー

音声ファイルとマーカーはすべて端末内にのみ保存されます。サーバーへのアップロードは行いません。
